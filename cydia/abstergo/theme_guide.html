Abstergo has two folder locations with themeable files.

A theming pack with all of Abstergo's images can be downloaded here.

/Library/Application Support/Abstergo

This folder contains Abstergo’s main files:

  • notification_icon.png
  • notification_icon@2x.png
  • remind_me_icon.png
  • remind_me_icon@2x.png

Bundle ID: com.arichardson.abstergo

/Library/PreferenceBundles/ AbstergoSettings.bundle

This folder contains Abstergo’s settings files:

  • AbstergoIcon.png
  • AbstergoIcon@2x.png
  • MailIcon.png
  • MailIcon@2x.png
  • TwitterIcon.png
  • TwitterIcon@2x.png

Bundle ID: com.arichardson.AbstergoSettings
